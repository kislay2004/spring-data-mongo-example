# Read Me First

 this project uses 'com.example.profileapplication' as main package.

# Getting Started
- run the app from ProfileApplication
- The app assumes a docker instance is running on localhost and port 27017
- once the app is running, use http://localhost:8080/profile/1dd8ff46-4e4c-49cd-90a9-34e8237e547b
- please replace the UUI from the one inserted in logs.
- swagger docs (is based on open api 3.0), once the app it can be accessed at: http://localhost:8080/api-docs
